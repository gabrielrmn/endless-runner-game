# COMO COMPILAR O JOGO : MEDIEVAL ROADS .

   - Para compilar o game é necessário estar com o terminal aberto na pasta do jogo ,se estiver no linux,
escrever no terminal :

                - cd caminho/para/ir/ao/diretorio/do/game (exemplo).
                - make run.

# COMO JOGAR (teclas e funcionalidades) O JOGO : MEDIEVAL ROADS . 

    - O jogo iniciará com a tela de menu aberta,deve-se selecionar a opcao START GAME para comecar a brincadeira.
    - Para isso , as teclas W e S movimentam o menu e a tecla ENTER escolhe a opcao desejada.
    - Caso queira ver as options e creditos é necessario manter a tecla ENTER pressionada.
    - Além disso as teclas W e S realizam o pulo e o agachar do personagem,lembrando que o agachar deve-se pressionar a tecla S.
    - A tecla P --> Pausa o jogo e abre o menu.
    - A tecla ESC --> Escolher se quer fechar o game ou continuar com ele aberto.
    - A tecla R --> Escolher se quer reiniciar o game ou continuar,caso deseje reiniciar,voltará ao marco inicial.
    - A tecla W --> Pulo do personagem.
    - A tecla S --> Agachar do personagem,mante-lá pressionada.
    - A tecla ENTER --> Mante-la pressionada para acessar options e credits,as demais,apenas um toque.

# COMO JOGAR (gameplay) O JOGO : MEDIEVAL ROADS .

    - Caso colidir , perde-se uma vida,começa com 3 vidas o player.
    - O item extra vida estará disponivel apenas quando o jogador estiver com menos de 3 vidas,evitando uma invencibilidade.
    - A moeda dá um boost de 100 pontos à pontuação.
    - A velocidade aumenta de acordo com a sobrevivencia do jogador.
    - De resto,pular e agachar :) .
